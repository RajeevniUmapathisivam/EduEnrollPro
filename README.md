# EduEnrollPro 
A full-stack web-based Course Registration System developed for university students and administrators.  
Students can register for semester courses while admins manage approvals, analytics, and course data.


##  Features
- Student Registration & Login
- Secure JWT Authentication
- Course Enrollment System
- Admin Approval Panel
- Dashboard Analytics
- Notifications & Alerts
- Responsive UI
- Separate Frontend & Backend Architecture


##  Technologies Used

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
